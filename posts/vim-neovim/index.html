Vim/Neovim

Vim commands for efficient text editing, including multiple line editing using Visual Block mode, removing search highlights, recording and applying macros, and resources for further customization, all aimed at enhancing productivity and streamlining the editing workflow.

Multiple Lines Edit

  • Visual Block ( Ctrl + v )
  • Select Char
  • Insert Mode (Capital I)
  • Esc

Remove Search Selection

  • Ctrl + l
  • :noh

Record

  • q - start
  • select register
  • q - stop
  • @register - apply
  • @@ - repeat last
