Some Linux Useful Commands

Convert files using convert

$ convert image.jpeg image.pdf

Open Files through terminal

$ xdg-open filename.png

pass yes argument without waiting response

$ yes | rm -r foldername/

usb– /media or /run/media

zip folder or file

$ zip -r <output> <input>

Size of File

du -h <filename>

list disk

# fdisk -l 

$ lsusb

eject / safely remove

$ eject /dev/<name>
